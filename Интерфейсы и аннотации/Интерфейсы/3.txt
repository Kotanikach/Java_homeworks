Практика #3 Объясните, какая область видимости у интерфейса, объявленного следующим образом.
protected interface Interface {
    // методы интерфейса
}

Область видимости: Интерфейсы не могут быть объявлены с модификатором protected вне класса. Если интерфейс объявлен в классе, то он будет доступен для подклассов этого класса и классов в том же пакете. Однако такое использование считается неправильным и приведёт к ошибке компиляции, если интерфейс находится вне класса.
