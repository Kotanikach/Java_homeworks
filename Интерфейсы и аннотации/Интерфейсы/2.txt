Практика #2 Объясните, какая область видимости у интерфейса, объявленного следующим образом.
public interface Interface {
    // методы интерфейса
}

Область видимости: Интерфейс с модификатором public доступен всем классам в любом пакете. Он может быть реализован любым классом, независимо от его расположения.
