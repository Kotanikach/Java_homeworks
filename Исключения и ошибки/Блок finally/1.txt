Практика #1
Можно ли использовать следующую конструкцию (try без catch и finally):

try {
  ...
}

Можно ли использовать конструкцию (try без catch и finally)
В Java нельзя использовать блок try без хотя бы одного блока catch или finally. Если вы попытаетесь написать код с только блоком try, это приведет к ошибке компиляции.
