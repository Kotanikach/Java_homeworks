Практика #2
Можно ли использовать следующую конструкцию (try только с finally):

try {
  ...
} finally {
  ...
}

Можно ли использовать конструкцию (try только с finally)
Да, в Java можно использовать блок try только с блоком finally. Это позволяет выполнять код в блоке finally, независимо от того, произошло ли исключение в блоке try. Однако исключение не будет обработано, если оно произойдет в блоке try.
