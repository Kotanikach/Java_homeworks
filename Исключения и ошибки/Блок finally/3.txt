Практика #3
Можно ли использовать следующую конструкцию (try с двумя finally):

try {
  ...
} finally {
  ...
} finally {
  ...
}

Можно ли использовать конструкцию (try с двумя finally)
Нет, в Java нельзя использовать два блока finally для одного блока try. Каждый блок try может иметь только один соответствующий блок finally. Если вы попытаетесь написать код с двумя блоками finally, это приведет к ошибке компиляции.
