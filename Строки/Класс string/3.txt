Практика #3 Зачем нужны три двойные кавычки?
Три двойные кавычки (""") используются для создания строковых литералов в некоторых языках программирования, таких как Python, для представления многострочных строк. Однако в Java это не поддерживается.

В Java для многострочных строк можно использовать символы переноса строки \n или использовать классы вроде StringBuilder или String.format() для более удобного форматирования многострочного текста.
