Оператор break: Оператор break завершает выполнение текущего цикла досрочно.
Пример:
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println(i);
}
Этот код выведет числа от 0 до 4, так как при i == 5 цикл прервётся.

Оператор continue: Оператор continue пропускает текущую итерацию и переходит к следующей.
Пример:
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue;
    }
    System.out.println(i);
}
Этот код выведет только нечётные числа (1, 3, 5, 7, 9), так как при чётных значениях i цикл пропускает вывод с помощью continue.

Пример использования цикла for для массивов и коллекций
Итерация по массиву с использованием цикла for-each:
int[] arr = {1, 3, 5, 7, 9};
for (int elem : arr) {
    System.out.printf("%d ", elem);
}
Этот код пройдёт по всем элементам массива arr и выведет: 1 3 5 7 9.
